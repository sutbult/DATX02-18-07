**Accept bid**
----
  Called when a user accepts a bid existing in the database and triggers the swapping process.

* **URL**

  /api/acceptBid

* **Method:**

  `POST`

* **Data Params**

  `{ "id": "RandomID" }`

* **Success Response:**

  * **Code:** 200 <br />
    **Content:** `{ }`

* **Sample Call:**

  `curl http://localhost:51337/api/addBid -d '{ "id": "RandomID" }'`
